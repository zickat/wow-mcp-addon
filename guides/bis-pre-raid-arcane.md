## Priorités stats
1. **Spell Hit** jusqu'au cap (202 / 16% sans Draenei dans le groupe)
2. **Spell Power** — stat reine Arcane
3. **Spell Crit** — synergies Arcane Concentration + Arcane Potency
4. **Intellect** — sustain mana en long fight

## Set Spellfire — Priorité absolue Tailoring 368

Le bonus **2/3 pièces** donne **+aura SP basée sur l'intellect** — énorme pour Arcane.

| Pièce | Matériaux clés | Pattern source |
|-------|----------------|----------------|
| Spellfire Robe | 16x Spellcloth, 4x Primal Might | Drop monde (Tailoring 375) |
| Spellfire Gloves | 8x Spellcloth, 2x Primal Might | Drop monde |
| Spellfire Belt | 8x Spellcloth, 2x Primal Might | Drop monde |

> **Note :** Tailoring 368 → pousser à 375 pour équiper. Spellcloth = cooldown hebdo (Netherstorm).
